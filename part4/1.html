第一节 GNSS北斗定位模块

Gravity GNSS 定位系统

SKU:TEL0157

DF商城

产品资料：Gravity GNSS 北斗定位模块

官方产品资料

1.简介

这是一款支持多卫星系统，可多系统联合定位和单系统独立定位的GNSS定位模块。能快速获取经纬度、时间、高度等数据。多系统联合定位相比于传统单一的GPS定位，可见和可用卫星数目大幅增加，能够提高定位精度和定位速度，即使是在复杂城市环境中也能实现稳定的高精度定位。

Gravity: GNSS定位模块，具有I2C和UART两种数据输出类型，兼容Arduino、ESP32、树莓派等主流的主控设备。可用于车载导航、手持定位、物品追踪、气象站等室外定位场景。

2. 产品参数

  • 工作电压： 3.3～5.5V DC
  • 输出信号： I2C/UART
  • 定位精度：2.0m CEP
  • 首次定位时间：冷启动30S 热启动2S
  • 功耗：40mA
  • 天线接口：IPEX1
  • 天线频率：1561-1575.42MHz±3MHz
  • 产品尺寸：27mmx37mm

3.引脚说明

标号 名称 功能描述
1 D/T I2C数据线SDA/UART数据传输-TX
2 C/R I2C时钟线SCL/UART数据接收-RX
3 - GND电源负极
4 + 电源正极

4.使用教程

Mind+1.7.3及以上版本，GNSS北斗定位模块的积木库。

  • 加载Mind+用户库：https://gitee.com/liliang9693/ext-gravitygnss

行空板测试代码

注意：I2C传感器，需要增加初始化pinpong的积木。

而使用行空板引脚积木时则不需要pinpong初始化积木。因为使用行空板引脚积木时默认会进行初始化操作，因此不再需要。

如果需要在行空板中直接运行python代码，直接把用户库中的 DFRobot_GNSS.py 文件放在同一目录下就可运行。

#  -*- coding: UTF-8 -*-

# MindPlus
# Python
from pinpong.board import Board
from DFRobot_GNSS import *
import time

Board().begin()
gnss = DFRobot_GNSS_I2C (bus=0, addr=0x20)
while (gnss.begin() == False):
    print("Sensor initialize failed!!")
    time.sleep(1)
gnss.enable_power()
gnss.set_gnss(GPS_BeiDou_GLONASS)
gnss.rgb_on()
gnss_utc = struct_utc_tim()
gnss_lat_lon = struct_lat_lon()

print("-------------------")

while True:
    time.sleep(3)
    gnss_utc = gnss.get_date()
    gnss_utc = gnss.get_utc()
    gnss_lat_lon = gnss.get_lat()
    gnss_lat_lon = gnss.get_lon()
    gnss_alt = gnss.get_alt()
    gnss_cog = gnss.get_cog()
    gnss_sog = gnss.get_sog()
    gnss_num = gnss.get_num_sta_used()
    gnss_star = gnss.get_gnss_mode()
    if ((int(float((str(gnss_num))))) > 3):
        print((str(gnss_utc.year)+'/'+str(gnss_utc.month)+'/'+str(gnss_utc.date)))
        print((str(gnss_utc.hour)+'/'+str(gnss_utc.minute)+'/'+str(gnss_utc.second)))
        print((str("经度：") + str((str(gnss_lat_lon.lonitude_degree)))))
        print((str("经度方向：") + str((str(gnss_lat_lon.lon_direction)))))
        print((str("纬度：") + str((str(gnss_lat_lon.latitude_degree)))))
        print((str("纬度方向：") + str((str(gnss_lat_lon.lat_direction)))))
        print((str("对地航速：") + str((str(gnss_sog)))))
        print((str("对地航向：") + str((str(gnss_cog)))))
        print((str("卫星数量：") + str((str(gnss_num)))))
        print("-------------------")
