第四节 带LED灯的数字按键

Gravity: 带LED灯的数字按键 蓝色

SKU:DFR0789

DF商城

这个LED灯的数字按键有蓝、黄、白、红、绿5种颜色。

官方产品资料

1.简介

LED Switch是一款按钮会发光的开关式按键。当我们按下按键时，它可以发出彩色，比如常见的“红、黄、绿、蓝、白”这五种颜色。同时它可以与micro:bit,Arduino等主板结合使用，然后实现很多有趣的互动作品。比如：开关、背光键盘、录播主控台、音乐播放台等与按键有关的作品。

2. 技术规格

  • 工作电压：+3.3-5V(推荐3.3V)
  • 输出类型:数字信号
  • 接口模式:PH2.0-3P
  • 尺寸：27×26.5mm
  • LED灯发光颜色：红、黄、绿、蓝、白

注：该模块带自锁功能。按下按键后，输出高电平，LED灯亮。再次按下后，输出低电平，LED灯灭。

3.引脚说明

序号 丝印 功能描述
1 D 控制端
2 + VCC
3 - GND

4.使用教程

传感器上标识D，数字接口，接行空板的P24（数字引脚）。

行空板测试代码

当按下按键，LED灯亮，行空板上显示的内容是：1； 当再按一次按键，LED灯灭，行空板上显示的内容是：0。

对照Python源码，看到读取数字引脚P24 这个模块后面是判断是否为True的，是的话结果是1，不是的话结果是0。所以在if 判断语句中，写不写判断 读取数字引脚P24是否等于1都可以。

#  -*- coding: UTF-8 -*-

# MindPlus
# Python
from pinpong.extension.unihiker import *
from pinpong.board import Board,Pin
from unihiker import GUI

u_gui=GUI()
Board().begin()
p_p24_in=Pin(Pin.P24, Pin.IN)
x=u_gui.draw_text(text="行空板",x=0,y=0,font_size=20, color="#0000FF")
print((str("数字引脚24：") + str((p_p24_in.read_digital()==True))))

while True:
    if (p_p24_in.read_digital()==True):
        x.config(text=(p_p24_in.read_digital()==True))
    else:
        x.config(text=(p_p24_in.read_digital()==True))
