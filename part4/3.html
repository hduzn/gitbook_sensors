第三节 USB摄像头

USB摄像头（兼容树莓派和Jetson Nano）

SKU:FIT0701

DF商城

官方产品资料

1.简介

这是一款插入USB端口就能使其工作起来的免驱摄像头，30W像素，640*480分辨率，即插即用，兼容树莓派和英伟达 Jetson Nano系列主板，特别在你的树莓派或 Jetson Nano主板装上外壳后使用起来更方便。

模块拥有30mm*25mm*21.4mm的小巧体积，可嵌入于各类设备，用于人脸识别，图像识别，安防，远程教学等项目之中。

2. 技术规格

  • 感光元件类型：CMOS
  • AGC / AEC /白平衡：自动
  • 红外滤镜：650±10nm
  • 支持压缩格式：MJPG，YUV
  • 固定图案噪声：< 0.03% of VPEAK-TO-PEAK
  • 模块接口：USB2.0
  • 像素：30W
  • 分辨率：640*480
  • 工作电压：5V
  • 工作温度：-20℃-70℃
  • 认证：FCC，CE
  • 尺寸：30mm*25mm*21.4mm
  • 支持操作系统：Windows，iOS，Android，Linux

3.使用教程

免驱，的以直接插行空板的USB接口上就可以用了。

行空板测试代码

1.显示摄像头实时画面

#  -*- coding: UTF-8 -*-

# MindPlus
# Python
import cv2

# 创建VideoCapture对象，0表示第一个摄像头，如果有多个摄像头，可以传入1, 2, ...
cap = cv2.VideoCapture(0)

cap.set(cv2.CAP_PROP_FRAME_WIDTH, 320)  #设置摄像头图像宽度
cap.set(cv2.CAP_PROP_FRAME_HEIGHT, 240) #设置摄像头图像高度
cap.set(cv2.CAP_PROP_BUFFERSIZE, 1)     #设置OpenCV内部的图像缓存，可以极大提高图像的实时性。

# 循环读取每一帧图像
while True:
    # 读取一帧图像
    ret, frame = cap.read()

    # 如果读取成功
    if ret:
        # 翻转画面，1表示水平翻转，为0表示垂直翻转，为-1表示水平和垂直都翻转。
        # frame = cv2.flip(frame, 1)

        # 显示图像
        cv2.imshow('frame', frame)

    # 按下q键退出循环
    if cv2.waitKey(1) & 0xFF == ord('q'):
        break

# 释放VideoCapture对象和窗口
cap.release()
cv2.destroyAllWindows()
