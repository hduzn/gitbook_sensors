第五节 数字大按钮模块

Gravity: 数字大按钮模块 黄色

SKU:DFR0789

DF商城

这个LED灯的数字按键有蓝、黄、白、红、绿5种颜色。

官方产品资料

1.简介

按压式的开关数字输入模块,与Arduino专用传感器扩展板结合使用,能够实现非常有趣的互动作品,按钮模块使用大按钮加优质按键帽手感一流,使用方便可以做到“即插即用”。

2. 技术规格

  • 工作电压：+3.3-5V
  • 模块自带指示灯，按下时会亮
  • 输出类型：数字信号
  • 尺寸：22*30mm

4.使用教程

传感器上标识D，数字接口，接行空板的P24（数字引脚）。

这个和前面的 带LED灯的数字按键 的区别是，带LED灯的数字按键带自锁功能，按下按键后，输出高电平，LED灯亮。再次按下后，输出低电平，LED灯灭；

而这个不带，意思就是这个按钮按下时是高电平，一抬起就是低电平了。

行空板测试代码

当按下按键，行空板上自带的LED（P25引脚）灯亮，显示的内容是：1； 当抬起手时，LED灯灭，行空板上显示的内容是：0。

#  -*- coding: UTF-8 -*-

# MindPlus
# Python
from pinpong.extension.unihiker import *
from pinpong.board import Board,Pin
from unihiker import GUI


u_gui=GUI()
Board().begin()
p_p24_in=Pin(Pin.P24, Pin.IN)
x=u_gui.draw_text(text="行空板",x=0,y=0,font_size=20, color="#0000FF")
print((str("数字引脚24：") + str((p_p24_in.read_digital()==True))))

while True:
    if ((p_p24_in.read_digital()==True) == 1):
        p_p25_out=Pin(Pin.P25, Pin.OUT)
        p_p25_out.write_digital(1)
        x.config(text=(p_p24_in.read_digital()==True))
    else:
        p_p25_out=Pin(Pin.P25, Pin.OUT)
        p_p25_out.write_digital(0)
        x.config(text=(p_p24_in.read_digital()==True))
